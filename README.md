# dl-papers

Third-party deep learning papers.

## Models

- [Wide ResNet](/dl_papers/wide_resnet)

## Guide

### Local installation

```
$ pip install -r requirements.txt
$ pip install -e .
```

### Docker container

```
$ docker build -t dl-papers .
$ nvidia-docker run --rm -it dl-papers /bin/bash
```

### Running models

See instructions with each model.
